Напишите код, который выводит элементы целочисленного массива a, у которых оба индекса четные, разделяя их в строке пробелами.

Считывать или создавать массив не нужно.

Верный по форме, но неверный по содержанию ответ:

System.out.print(a[0][0]);

Для проверки Вашего кода на примере ниже, он будет вставлен в следующий шаблон:

public class Main {
    public static void main(String[] args) {
        int[][] a = {{1, 2, 3},
                     {4, 5, 6},
                     {7, 8, 9} 
                    };
        // Ваш ответ будет вставлен сюда
    }
}

Ответ:
for (int i = 0; i < a.length; i += 2) {
    for (int j = 0; j < a[i].length; j += 2) {
        System.out.printf("%d ", a[i][j]);
    }
    System.out.printf("\n");
}
