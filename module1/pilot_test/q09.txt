В коде объявлены и проинициализированы целые переменные x и c. Перепишите фрагмент кода

while (c < 50) {
     if (x >= 500) break; 
     c++;
     x += c * 2;
}  

на следующий оператор for (допишите нужный текст в скобки):

for (/* Ваш ответ будет вставлен сюда */) {
    // Тело цикла пустое
}

Ответ:
; c < 50 && x<500; c++, x+=c*2
